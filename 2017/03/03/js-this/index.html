解题想法

我的理解，this其实跟其他语言的self应该是差不多。就是指代自己，比如说在一个function里面，那么this指向的就是这个func。

可能我讲的不清楚，就是this其实就是括号里的东西。

当然这是我的理解，我觉得应该就是这样的，所以刚刚那样取到的this并不是this。
