2019 总结

这篇文章诞生于支离破碎的主题中，因为博客主题正在改造，很长时间没更新也是因为改动太大完全收拾不开..

—— 2018 年终总结

翻看了往年的年终总结，发现每次这个时候都在改博客…甚至有种没改完懒得写博客的心态…

博客风格更新

重构了主题，对颜色搭配重新定义了一下：灰阶 + 低明度的主色以不同比例分配。因此，不仅可以让每位用户都可以设置属于自己的主题，同时我可以每年都以潘通年度流行色来变化自己的博客……

2020 年流行色是经典蓝，呐

前期卖血吃利息

我经历了接近半年没有稳定工作的生活，调整了理财计划，所以不至于饿死。

期间面试了各种冷门中大厂，吃尽各种亏，回想起来都觉得去年去大疆那次太简单了。但也不是没有收获，至少我学会了应该怎么写简历，什么是“互联网”，大厂面试都在问什么。

所以一路疯狂卖血吃利息，最后去了一家技术型公司，总算目标达成。下意识的回想了一下，今年竟然没有抱怨任何，心态也算成长了吧。

荒废技术

我自己已经感受到我离大厂已经越来越远了，由于心态变化也减少了很多念想。可能有机会，但应该很难胜任技术岗。

现在在带前端小团队，同时做一个项目的项目管控，做了太多关于项目管理的工作，工作主要技能点也都在点资源分配、流程优化之类的工作。

然后依然每年传统艺能：一个人一支军队做一个项目……

技术工作也大多是内容管理，休息时间要不是觉得太累，就是技能树乱点。总的来说就是：前端没有继续深入学习，离资深前端又远了一步。

开源乱葬岗

GitHub 基本没有产出，有也是公司组件维护。（欢迎Vue技术栈试试，提升外包速度（X

而且今年工作都是用公司电脑（心愿达成x2（19Early顶配白买（垃圾库克出16背刺👴)))，由于工作环境变成了 Windows 所以有一半工具链无法使用，也懒得摸鱼也没有特别充足的时间摸鱼。

算是产出了一个能用的东西：Resumer，用来生成能用的简历，为自己面试准备的。时间上比较亏，因为写个简历没多久，改简历也没多久，写个工具耗费的时间完全补不回来。不过有些面试倒是起作用了，例如描述业余项目。

如果算的话还有一个，重构了Nlvi（年更主题，年年 breaking change

书和电影

书是这些：

  • 《旅行摄影圣经》
  • 《沟通心理学》（别读，浪费时间）
  • 《如何读一本书》
  • 《遇见未知的自己》
  • 《告别失控：软件开发团队管理必读》
  • 《苏菲的世界》
  • 《高效程序员的45个习惯》
  • 《穗子的动物园》

《穗子动物园》应该是读得最嗨的书，书出自严歌苓老师之手。总之文学小说还是自己读才能感受到魅力。
《遇见未知的自己》直接把我对未来的恐惧全部打掉，读完后对自己人生的看法发生了转变 —— 从「为什么我去不了大厂」到「我不配去大厂，但也不错」。

修了一个网课：《微观经济学》，临近结课的时候修不下去了

电影：

  • 《天气之子》
  • 《冰雪奇缘2》
  • 想不起来

关于音乐

今年听了很多类型的音乐，EDM 的流派基本都摸清楚了。最近也在复习乐理，然后做做 future bass

音乐跟编程真像啊，就像玩 future bass 一样，大家都在探索自己的音色和自己的鼓组搭配，带 DJ 也会调出自己的音色或者鼓组打包成套以便以后的编曲，或以此谋生。

像极了程序员的模块组件思维……

反思

所以出现了一些问题：

  • 时间管理
  • 技术成长

对此采取一点策略：

  • 把一年想做的列出清单，分季度做完，并且保证做完不要拖延
  • 实现的时候还是用上番茄钟，坐太久身体不好又容易烦躁
  • 开始规划做 leetcode，不过先找算法入门的书看看（总卡在这）
  • 副系点了Rust，那么专注点完
  • 博客继续更新，不再写划水文，例如脚手架使用什么的
